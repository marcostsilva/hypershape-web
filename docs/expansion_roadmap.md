# HyperShape — Roadmap de Expansão (Pós-ERP)

Este documento detalha as funcionalidades de expansão planejadas para após a conclusão da base do ERP.

## 1. Expansão Prioritária (Fase 5)

### A. Ecossistema White-Label (Item 3)
- **Customização de Marca:** Interface administrativa para que cada academia defina sua identidade visual (Logo, Cores Primárias/Secundárias).
- **Subdomínios/Domínios Customizados:** Suporte para que academias usem `suaacademia.hypershape.com` ou domínios próprios.
- **Experiência Customizada:** Adaptação dinâmica de componentes UI baseada no `primaryColor` da unidade.

### B. Gamificação e Engajamento (Item 4)
- **Leaderboards Locais:** Rankings de consistência e desempenho dentro de cada academia.
- **Sistema de Conquistas (Badges):** Medalhas virtuais por marcos atingidos (ex: Consistência, Progressão de Carga).
- **Feed Social da Unidade:** Mural de avisos e interações entre alunos e instrutores da mesma unidade.

## 2. Módulo Financeiro (Foco Inicial: B2B)
- **Controle de Assinaturas HyperShape:** Sistema para gerenciar o pagamento das academias para a plataforma.
- **Billing Portal:** Histórico de faturas e gestão de cartões/métodos de pagamento para Gym Admins.
- **Upgrade de Tier Automático:** Facilitação de migração entre planos (Starter -> Pro -> Elite) conforme o limite de alunos é atingido.

## 3. Sugestões para Futuro Remoto (Backlog)
- **Financeiro B2C:** Cobrança da academia para o aluno.
- **AI Coach:** Prescrição autônoma baseada em dados.
- **IoT/Catracas:** Integração física com academias.
- **Wearables:** Apple Health / Google Fit.
